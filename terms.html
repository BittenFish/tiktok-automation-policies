Términos de Servicio - Script de Automatización de TikTok

Última actualización: 16 de Julio de 2025

Este documento describe los términos de uso para el script de automatización de TikTok (en adelante, "el Script"), una herramienta de uso personal diseñada para automatizar la subida de contenido de video a la plataforma TikTok.

1. Naturaleza del Servicio

El Script es una herramienta de software diseñada para ser ejecutada localmente por el usuario. Su propósito es facilitar la gestión personal de contenido en TikTok mediante la automatización de las cargas de video. No es una aplicación comercial ni un servicio en la nube ofrecido a terceros.

2. Uso Personal y No Comercial

El Script está destinado exclusivamente para uso personal y no comercial. El usuario es el único responsable de la configuración, ejecución y el contenido que sube a TikTok a través del Script.

3. Privacidad y Datos

El Script no recopila, almacena, ni comparte información personal del usuario ni de su cuenta de TikTok más allá de lo estrictamente necesario para interactuar con la API oficial de TikTok (como tokens de acceso temporales para autenticación). Todos los videos se procesan y suben directamente a la cuenta de TikTok del usuario sin que el Script almacene copias en servidores externos.
