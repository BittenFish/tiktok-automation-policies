Política de Privacidad - Script de Automatización de TikTok

Última actualización: 16 de Julio de 2025

Esta Política de Privacidad describe cómo el script de automatización de TikTok (en adelante, "el Script") maneja la información del usuario. El Script es una herramienta de software diseñada para uso personal y local, cuyo objetivo principal es automatizar la subida de contenido de video a la plataforma TikTok.

1. Recopilación y Uso de Información

El Script está diseñado para funcionar sin recopilar ni almacenar información personal identificable del usuario. La interacción con la API de TikTok se limita a lo esencial para la autenticación y la subida de contenido.

  • Autenticación: El Script utiliza tokens de acceso y refresh tokens, obtenidos a través del proceso de autenticación OAuth de TikTok, para interactuar con la API en nombre del usuario. Estos tokens son gestionados de forma local por el usuario y no son almacenados en servidores externos controlados por el creador del Script.
  • Contenido de Video: Los archivos de video que el usuario desea subir se acceden directamente desde la computadora local del usuario y se transfieren a TikTok a través de su API oficial. El Script no almacena copias de estos videos ni los procesa en servidores de terceros.
  • No Recopilación de Datos Personales: El Script no está diseñado para recopilar nombres, direcciones de correo electrónico, números de teléfono ni ninguna otra información personal del usuario.

2. Almacenamiento de Datos

Los únicos datos que el Script puede almacenar localmente son los tokens de acceso y refresh tokens necesarios para la persistencia de la sesión de la API, y un registro (log) de los videos ya subidos para evitar duplicidades. Estos datos se guardan únicamente en la máquina del usuario donde se ejecuta el Script.

3. Divulgación de Información

El Script no comparte ni divulga ninguna información del usuario con terceros. Toda la interacción es directamente entre el Script (ejecutándose en la máquina del usuario) y la API oficial de TikTok.

4. Seguridad

La seguridad de su información depende de la seguridad de su propia computadora y de cómo maneje los tokens de acceso generados. Se recomienda mantener su sistema operativo y su software actualizados.

5. Cambios a esta Política de Privacidad

Esta Política de Privacidad puede ser actualizada ocasionalmente para reflejar cambios en la funcionalidad del Script o en los requisitos de la API. Se recomienda al usuario revisar esta política periódicamente.

6. Contacto

Si tiene alguna pregunta sobre esta Política de Privacidad o sobre cómo el Script maneja la información, por favor, contacte directamente al creador del Script.

Al utilizar este Script, usted acepta las prácticas descritas en esta Política de Privacidad.
